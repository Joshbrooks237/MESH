## How to Use

1. **Open** `fun_particle_toy.html` in any modern web browser
2. **Move your mouse** around to attract existing particles
3. **Click and hold** to create new particles that follow your cursor
4. **Drag while holding** to paint beautiful particle trails
5. **Experiment** with different keyboard controls for various effects

## Technical Details

- Built with vanilla HTML5, CSS3, and JavaScript
- No external dependencies required
- Responsive canvas that adapts to window size
- Optimized particle system handling up to 800 particles
- Smooth 60fps animation with requestAnimationFrame
