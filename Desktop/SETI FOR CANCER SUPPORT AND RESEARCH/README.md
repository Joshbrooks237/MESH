### Prerequisites
- Node.js 18+
- Python 3.9+

### Installation

```bash
# Clone the repository
git clone https://github.com/seti-cancer-research/universe.git
cd seti-cancer-research

# Install dependencies
npm install
pip install -r requirements.txt

# Start development environment

# Run the application
npm run dev
```
